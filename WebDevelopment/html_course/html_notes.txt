-> What is HTML??
    - HTML full form is Hyper Text Markup Language
    - Used to create the structure of Web Pages

    - Web Page: a Page which consists of different formats of Content
    - Content: Text, Images, Media Objects: Video, Audio, Links, Inputs etc., 


-> How to Create the structure of the Web Page??
    - To Create the structure of the Web Page
        1. Elements: 
            1. To represent that particular format of a Content
            2. Types of Elements:
                1. Block
                    syntax:
                    <elementName>
                        .....
                        .....
                    </elementName>
                    examples:
                    <p>....</p>, <div>...</div>, <h1>...</h1>, ....

                2. Inline
                    syntax:
                    <elementName 
                        ...
                        ...
                    />
                    examples:
                    <input />, <hr />, <img />, <br />


        2. Attributes
            - An attribute is used to add the characteristics or specification to the elements
                1. Block
                    syntax:
                    <elementName attributeName='itsValue' attributeName>
                        .....
                        .....
                    </elementName>
                    examples:
                    <p style=''>....</p>, <div hidden>...</div>, <h1>...</h1>, ....

                2. Inline
                    syntax:
                    <elementName attributeName='' attributeName 
                        ...
                        ...
                    />
                    examples:
                    <input type='' />, <hr />, <img src='' />, <br />'
