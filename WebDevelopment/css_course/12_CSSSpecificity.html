CSS Specificity

CSS Specificity: refers to the set of rules that determines which styles are applied when they are conflicting or ovelapping style declaration.

It determines which style rule takes precedence over others and how the final rendering of an element is determined

CSS Specificity is based on the combination of different selectors used in a style rule. Each selector carries a certain weight and the selector with higher specificity. The specificity is calculated based on the following criteria:

  • Inline Styles: Styles defined directly on an HTML element using the `style` attribute. They have high specificity
  • ID's: Seletors targeting elements by their unique ID attibute using the symbol `#`. They have a higher specificity than classe's and tag's
  • Classe's, Attributes and PsuedoClasse's: Selectors targetng elements based on Class Names, Attributes, PsuedoClasses. They have the lowest specificity
  • Elements and PsuedoElements: Selectors targeting elements based on their TagName's (eg., div, p) or PsuedoElements (eg., ::after, ::first-child). They have the lowest specificity
